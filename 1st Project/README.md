# Description of Project

This project consisted in developing a set of web UIs for a (nonexistent) food company that sells fresh dishes. The team developed UI designs with Figma and Adobe Illustrator, as well as the static website using HTML, CSS and SASS pre-processor.  
_Team for this project: [@prncjs](https://github.com/prncjs) [@jplaudir8](https://github.com/Jplaudir8)_
